Игра Угадайка

Игрок загадывает любое целое число в выбранном диапозоне, а Робот его угадает.

Правила игры

  1. минимальное и максимальное значения задаются игроком в начале игры
  2. наименьшее число, которое можно указать для диапазона -999
  3. наибольшее число, которое можно указать для диапазона 999
  4. загаданное число должно находиться внутри заданного диапазона и должно быть целым
  5. ответы робота корректирует игрок с помощью кнопок "больше" и "меньше"
